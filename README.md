# 🌐 Web Development Internship - SkillCraft Technology

This repository showcases the projects I developed during my Web Development Internship at **SkillCraft Technology**. These tasks focused on building responsive layouts, implementing time-based logic, and creating interactive user interfaces.

## 📌 Projects Included

### 🎨 1. Responsive Landing Page
A modern and visually appealing landing page with interactive elements and high-quality hover effects.

🔹 **Features:**
* **Responsive Design:** Fully optimized for all devices (Mobile, Desktop).
* **Interactive Hover Effects:** Smooth CSS transitions on navigation and call-to-action buttons.
* **Modern UI:** Clean layout using CSS techniques.

🔹 **Technologies Used:**
* HTML, CSS, JavaScript

### ⏱️ 2. Stopwatch Web Application
A functional and precise stopwatch application for tracking time intervals.

🔹 **Features:**
* **Start, Pause, and Reset:** Core timing functionality.
* **Lap Recording:** Ability to record specific time marks during a session.
* **Clean Interface:** User-friendly and minimalist design.

🔹 **Technologies Used:**
* HTML, CSS, JavaScript

### 🧠 3. Interactive Quiz Game
A dynamic quiz application that tests user knowledge and tracks performance in real-time.
